# Darajani — Next.js Website

Premium tourism website for **Darajani Motel, Camping & Tour Operations** (Sanya Juu, Siha, Kilimanjaro Region, Tanzania).

## Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **Framer Motion** (scroll and transition animations)
- **next/image** for optimized images

## Design

- **Palette:** Forest Green `#1F3D2B`, Warm Brown `#7A5230`, Sand `#F6F1E9`, Charcoal `#1A1A1A`
- **Typography:** Playfair Display (headings), Lato (body)
- **Tone:** Authentic, warm, earthy, premium, cultural and mountain-inspired

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Assets

Images live in `public/assets/` (copied from the design assets). Update WhatsApp number and email in:

- `src/components/Hero.jsx`
- `src/components/Booking.jsx`
- `src/components/Footer.jsx`

## Build

```bash
npm run build
npm start
```
